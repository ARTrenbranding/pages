What is go.artren.kr?

go.artren.kr는 어떤 페이지인가요?

go.artren.kr는 ARTren이 클라이언트별 자료를 전달하기 위해 사용하는 전용 게이트웨이입니다. 공식 홈페이지에서는 서비스와 포트폴리오를 소개하고, 이 보조 도메인에서는 프로젝트별 리포트, 제안서, 검수 페이지, 결제 안내 등 실행 자료를 전달합니다.

Why this gateway exists

링크 하나에도 브랜드 경험이 필요합니다

ARTren은 단순히 파일을 전달하지 않습니다. 클라이언트가 받은 자료가 어떤 목적을 가지고 있고, 다음에 무엇을 확인해야 하는지 한눈에 이해할 수 있도록 프로젝트별 페이지로 정리합니다.

브랜드 콘텐츠는 제작으로 끝나지 않습니다.
전달, 검수, 리포트, 다음 제안까지 하나의 경험으로 설계되어야 합니다.

전달받은 링크는 이렇게 확인하세요

ARTren이 전달하는 모든 프로젝트 페이지는 동일한 순서로 확인할 수 있습니다.

01

링크 주소 확인

ARTren이 전달하는 프로젝트 링크는 보통 go.artren.kr/프로젝트명/ 형태입니다.

02

자료 목적 확인

리포트, 제안서, 검수, 결제 안내 중 어떤 목적의 페이지인지 먼저 확인합니다.

03

내용 검토

페이지 안의 안내에 따라 자료를 확인합니다.

04

수정 또는 문의 전달

확인 후 필요한 수정, 질문, 다음 진행 여부를 ARTren 담당자에게 전달합니다.

개별 프로젝트 페이지는 클라이언트별 자료이므로 외부 공유를 권장하지 않습니다.

Security & Privacy Notice

클라이언트 자료 보호 기준

ARTren은 클라이언트별 리포트와 제안서를 프로젝트 단위로 분리해 전달합니다. 공개 페이지에는 클라이언트명, 계약 조건, 성과 데이터, 내부 자료를 노출하지 않습니다.

  • 공개 페이지에 클라이언트 실명 노출 최소화
  • 리포트별 URL 개별 관리
  • 민감 자료는 별도 전달
  • 검색 노출이 불필요한 페이지는 noindex 적용 가능
  • 결제 링크는 필요한 경우에만 개별 전달
